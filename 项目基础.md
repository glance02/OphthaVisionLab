# 项目基础

本项目的指导教师徐晓长期从事基于人工智能的糖尿病视网膜病变（DR）智能诊断相关研究。目前，徐晓老师主持国家自然科学基金青年项目“面向医疗数据的深度密度聚类算法研究”、南开大学眼科学研究院开放基金“基于深度学习的病理性近视智能诊断系统研究”等5项医工结合科研课题，并参与国家自然科学基金面上项目、江苏省自然科学杰出青年基金项目、江苏省自然科学青年基金项目等4项项目，在相关领域积累了丰富的研究经验。

在上一次校级大学生创新创业训练计划项目中，我们团队围绕眼科图像智能分析完成了较系统的研究与原型开发工作，已经形成了可直接支撑本次申报的前期成果。前期项目以 VisionFM 为重要基础，完成了相关代码框架的梳理、本地部署与任务适配，保留了预训练、微调、特征提取、分类与分割等完整技术链路，并结合眼底图像分析需求，建立了面向血管分割、疾病分类等任务的实际应用流程。通过前一阶段的工作，团队已经掌握了眼科基础模型在本地环境中的训练、迁移和推理方法，为后续更复杂系统的构建打下了稳定基础。

在算法层面，上一轮项目不仅应用了 VisionFM 的通用眼科视觉表征能力，还完成了针对微动脉瘤分割任务的专项模型整合与应用。现有仓库中已经包含面向 IDRiD 微动脉瘤分割的模型结构、图像预处理、patch 级推理、病灶连通域统计等完整实现，同时保留了 VisionFM 框架下的眼底血管分割、二分类、多分类、多任务训练等模块。前期项目已经能够对眼底图像实现血管分割、微动脉瘤分割、疾病分类和结构化结果输出，形成了“通用基础模型 + 专项病灶模型”相结合的算法体系。这一成果表明，团队前期研究已经从单一模型应用推进到面向多任务、多层次眼科影像分析的综合建模。

在系统实现层面，上一轮项目已经完成了较完整的眼科智能分析原型系统建设。后端基于 FastAPI 封装了统一的多任务推理服务，能够提供血管分割、微动脉瘤分割、二分类、多分类和 AI 智能分析等接口；前端基于 Vue 3 构建了图像上传、参数设置、结果展示和报告下载界面；系统同时具备 Windows 与 Linux 环境下的启动与演示能力。除此之外，前期项目还开展了本地模型结果与多模态大模型结合的探索，可在分割和分类结果基础上提取结构化特征，并进一步生成面向医生参考的自然语言分析报告。由此，我们团队已经积累了从模型能力、接口服务到可视化交互的完整原型开发经验。

本次申报的“多模态大模型驱动的眼科多智能体会诊辅助系统研究”，正是在上述校级大创成果基础上的进一步提升。前期项目已经完成了眼科图像多任务分析能力建设和原型系统搭建，本课题将在此基础上继续向多模态输入、临床信息融合、智能体分工协作和会诊辅助决策方向拓展，把已有的血管分割、微动脉瘤识别、疾病分类和报告生成能力进一步组织为可协同工作的多智能体系统，并引入病历、检查结果、知识库等信息源，形成面向临床场景的眼科多智能体会诊辅助平台。因此，上一轮校级大创项目已经为本课题提供了扎实的模型基础、系统基础和应用基础。
